## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/felipealvescosta/joker
```

2. Install packages

```sh
yarn install
```

3. Run Project

Run Server

```sh
yarn dev
```

Run Mobile App

```sh
expo start
```
